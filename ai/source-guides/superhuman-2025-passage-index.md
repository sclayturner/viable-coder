# Superhuman 2025 Passage Index

This index makes selected passages from `archive/prior-books/superhuman-2025.md` usable as historical source material for the current Viable Coder playbook.

The archived book remains provenance only. Do not edit it. Do not import passages wholesale. Use this index to find durable founder, engineering, and personal-life source material, then translate it into current Viable Coder doctrine before promoting it into `book/`, `doctrine/`, `research/`, `website/`, or harness guidance.

## Why This Exists

The existing source guide maps the prior book by chapter. That is useful for orientation, but too coarse for drafting. The playbook needs passage-level source handles: moments that can be routed to a concept, a reader situation, an ADLC gate, or a long-game signal.

This file is the missing middle layer. It does not make the prior book canonical. It makes the archive searchable by use.

## Index Fields

- **Passage ID**: stable handle for reference in notes or drafting tasks.
- **Source lines**: line references in `archive/prior-books/superhuman-2025.md`.
- **Experience source**: founder, engineering, personal life, or combination.
- **Current playbook use**: where the passage can support the current architecture.
- **Why it matters**: the durable idea beneath the old frame.
- **Promotion note**: what must happen before use in publishable or canonical material.

## Promotion Rules

1. Treat every passage as historical reference until promoted elsewhere.
2. Translate away from "superhuman" and "Soaring Titan" language unless discussing historical provenance.
3. Preserve lived answerability: the human keeps origination, judgment, review, and consequence.
4. Verify dated AI, market, political, product, or provider claims before using them as current evidence.
5. Use personal material only when it does structural work for the reader. Do not use it as decorative memoir.
6. When a passage supports doctrine, reconcile terms with `doctrine/playbook-lexicon.md` before promotion.
